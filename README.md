# iTechnoPro MVP 25 Vibe
MVP веб-приложения для автоматизации аутсорсинга охраны труда и промышленной безопасности компании АйТехноПро.
Создано в октябре 2025 года в рамках обучения Вайб-Кодингу (ChatGPT Codex).

## Как проверить демо "Safety Runner"

1. Склонируйте репозиторий или скачайте архив.
2. Откройте файл `index.html` двойным щелчком в браузере **или** поднимите простой сервер:

   ```bash
   python -m http.server 8000
   ```

   Затем перейдите в браузере на `http://localhost:8000`.
3. Управление в игре: ← → для смены полосы, ↑ или Space для прыжка, `E` — действие рядом с объектом. Счет отображается в правой панели.
